# Bank Value Overlay Plugin
Calculates your entire character's bank, inventory, and equipment gp value. Displays the overall value of your account on the screen as an overlay.

## Configuartion options

![img_3.png](screenshots/Configuration.png)

### Show Suffix
If unchecked, the plugin will remove the K/M/B suffix from the displayed values, and displays a more precise value.

### Planned Features
- Show GE value
- Include containers in calculations (ex. gem bag, looting bag, log basket, etc.)
- Include options to show specific containers separately (ex. looting bag)
- Include potion storage in bank calc
- Showing the last bank value from your last session
